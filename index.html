Features

Everything a parent needs,
nothing a child finds confusing

On-device Gemma 4 2B

The full Gemma 4 2B model runs entirely on your phone via LiteRT. Your child's schedule stays private — always.

Voice input

Tap the mic and speak your plan in any supported language. Locale-aware speech recognition switches automatically.

50+ activity types

Car, bus, school, park, eat, sleep, doctor, shop, bath, and many more — each with a clear icon and colour.

7 languages

English, Swedish, Spanish, German, French, Portuguese, Arabic. Cards and the full UI switch language together.

GPU acceleration

Uses GPU inference by default with automatic fallback to CPU — fast response even on mid-range Android devices.

Clean iOS-style UI

Big cards, clear icons, smooth animations — designed so children can follow along without needing to read.

How it works

From words to visual cards
in seconds

1

Describe the day

Type or speak a free-text plan in any of the 7 supported languages. No special formatting needed.

→
2

AI parses the plan

The on-device Gemma model processes your text and returns a structured JSON array of activity steps.

→
3

Cards appear instantly

Each step is matched to an icon and colour from the built-in library and displayed as a big visual card.

Sample input → output
You say: "Drive to school, then walk to the park, then home for lunch and bath"
↓ Gemma 4 2B (on-device)
[
  { "label": "Drive to school", "icon": "car"   },
  { "label": "Park",            "icon": "park"  },
  { "label": "Lunch",           "icon": "eat"   },
  { "label": "Bath time",       "icon": "bath"  }
]

Requirements

  • Flutter 3.x
  • Android (ARM64, Android 7.0+ / API 24+) or iOS
  • ~2.5 GB free storage (for the model file)
Open source

Get started in minutes

Clone, run, and download the model in-app. One-time 2.4 GB download, then works fully offline forever.

1 flutter pub get && flutter run
2 Tap Settings icon → Download Gemma 4 2B
3 Wait for the ~2.4 GB download (one-time only)
4 Done — works fully offline from here on
